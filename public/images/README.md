# Images Folder

Drop your images here and they will be mapped to the portfolio.

## Required Images:

| Filename | Usage | Recommended Size |
|----------|-------|-----------------|
| `profile.jpg` | About section - your profile photo | 400x400px |
| `project-parking.jpg` | Vehicle Parking Management project | 600x400px |
| `project-stress.jpg` | Stress Management Website project | 600x400px |
| `project-translate.jpg` | Vision Translate project | 600x400px |
| `project-spam.jpg` | SMS Spam Detection project | 600x400px |
| `project-movie.jpg` | Movie Recommendation project | 600x400px |
| `football.jpg` | (Optional) Football hobby image | 400x300px |
| `bike.jpg` | (Optional) Bike riding hobby image | 400x300px |

## Notes:
- Images should be in JPG or PNG format
- Keep file sizes under 500KB for fast loading
- The portfolio gracefully handles missing images with gradient placeholders
