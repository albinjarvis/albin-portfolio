# Images Folder

Drop your profile photo here and it will be mapped to the portfolio.

## Required Image:

| Filename | Usage | Recommended Size |
|----------|-------|-----------------|
| `profile.jpg` | Your photo in the About section | 400x500px (3:4 ratio) |

## Notes:
- Use JPG or PNG format
- Keep file size under 500KB for fast loading
- The portfolio shows a stylish "AJ" fallback if no image is found
- No other images needed - the portfolio uses abstract gradients and icons
